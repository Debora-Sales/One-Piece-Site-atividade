# Projeto One Piece - Atividade SENAC (UC15)

Este repositório contém o projeto de conclusão da Unidade Curricular 15 (UC15) do curso de Tecnologia no SENAC. O objetivo do trabalho foi desenvolver um site com temática livre, aplicando conceitos de estilização profissional, interatividade e organização de arquivos.

## 🌐 Demonstração
 **O site está publicado e pode ser acessado aqui 👉 [link](https://debora-sales.github.io/One-Piece-Site-atividade/)**

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação das páginas.
* **CSS3:** Estilização customizada (`style.css`).
* **JavaScript:** Implementação de lógica para formulários e interações dinâmicas.
* **Bootstrap 5:** Framework para layout responsivo, componentes de navegação, cards e botões.
* **Bibliotecas Externas:** Utilização de Bootstrap Icons e recursos de efeitos visuais (Parallax).

## 📂 Estrutura do Projeto
O site é composto por diversas páginas interconectadas, mantendo uma identidade visual padronizada:
* `index.html`: Página inicial com introdução ao tema.
* `historia.html`: Detalhes sobre a narrativa do anime.
* `tripulacao.html`: Apresentação dos personagens com interações de vídeo/foto.
* `criador.html`: Informações e curiosidades sobre Eiichiro Oda.
* `contato.html`: Formulário de contato funcional com validação via JavaScript.
* `spoilers.htmll`: Conteúdos adicionais para fãs.

## 🛠️ Funcionalidades em Destaque
1.  **Design Responsivo:** Adaptado para diferentes tamanhos de tela através do sistema de grid do Bootstrap.
2.  **Multimídia:** Integração de fotos e vídeos para uma experiência imersiva.
3.  **Formulário de Contato:** Implementação de captura de dados e feedback visual para o usuário.
4.  **Organização de Código:** Separação clara entre estrutura (HTML) e estilo (CSS), com comentários explicativos para fins didáticos.

## 🎓 Contexto Acadêmico
Este projeto foi desenvolvido como parte integrante das aulas no SENAC, focado em demonstrar competência em:
* Criação de menus de navegação complexos.
* Uso de classes utilitárias de framework.
* Manipulação de DOM e requisições assíncronas simples (fetch).
* Manutenção de identidade visual consistente em múltiplas páginas.
